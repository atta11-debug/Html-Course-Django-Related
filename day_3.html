Learning about Lists and Tables

Lists are used to group related items together.

Unordered List (ul) :
Displays items with bullets. Like :

  • Apple
  • Banana
  • Mango
  • Orange
Ordered List (ol) :
Displays items with numbers (1,2,3). Like :
  1. Atta Ul Mustafa
  2. Amna Ramzan
  3. Mudassir Ijaz
  4. Sara Iqbal

Nested List :

  • Fruits
    1. Apple
    2. Mango
  • Vegetables
    1. Lady Finger
    2. Potato
