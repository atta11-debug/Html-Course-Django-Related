Semantic HTML & Layout

  1. Semantic HTML Elements

    • "header"

      Represents the top section of a page or article.
      Usually contains logo, navigation, title, search bar.
      For Example :

      My Website

      Learning html step by step.

    • "nav"

      Defines a navigation menu (links to other pages or sections).

      Youtube| Google| GitHub

    • "main"

      The main content area of the page.
      Only one main per page (unique).

      Welcome to My Site

      This is where the main content goes.

    • "section"

      Groups related content together into sections.
      Example: About section, Services section, Contact section.

      About Me

      I am learning Django and Web Development.

      School

      Govt. Boys Higher Secondary School Bagh.

    • "article"

      Represents independent content (like blog post, news article, forum post).
      Can stand alone if taken out of context.

      Latest News

      Django 5.0 has been released with new features!

    • "footer"

      Bottom section of page/article.
      Contains contact info, copyright, links, social media.

      © 2025 Atta’s Learning Journey

  2. "div" vs "span"

    Both are non-semantic (don’t add meaning, just structure/styling).
    'div' = block-level container (big boxes, layouts).
    Like :

    This is a div (block box).

    'span' = inline container (small text parts inside a line).

    This is a highlighted word inside a sentence.
