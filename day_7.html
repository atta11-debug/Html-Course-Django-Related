Global Attributes

Global attributes are special attributes that can be added to almost any HTML tag. They control identification, styling, behavior, and accessibility.

Common Global Attributes:

  • id → Unique identifier for an element (used in CSS/JS).

    This is an introduction.

  • class → Groups elements for styling with CSS.

    This is highlighted text.

  • style → Adds inline CSS directly.

    Hello World

  • alt → Alternative text for images (for accessibility & when image can’t load).
  • title → Shows tooltip text when hovering.

HTML Entities

HTML entities are special codes that represent reserved characters or symbols in HTML. Because some characters (like < and>) are used in HTML tags, we can’t write them directly.
✅ Common Entities:
< → < (less than sign)
> →> (greater than sign)
© → © (copyright symbol)
  → (non-breaking space, adds extra space)
" → " (double quote)
& → & (ampersand)
EXAMPLE:

Use <h1> for the largest heading.
